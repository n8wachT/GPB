# GPB
Generic Python3 Bot.

This bot uses [pyTelegramBotAPI](https://github.com/eternnoir/pyTelegramBotAPI).

Install it with :
```
sudo pip install pyTelegramBotAPI
```

And clone the bot:
```
git clone https://github.com/sanguchi/GPB/
```
Don't forget to edit the token variable in Bot.py
